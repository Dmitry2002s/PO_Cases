## Ключевые кейсы (всё с цифрами)

- **[Task Extractor](./task-extractor)** – обработка Excel-задач в единую таблицу со сквозным приоритетом. Экономия **25-30 человеко-часов в месяц** для тимлидов и руководителей.

- **[Financial Reports](./financial-reports)** – автоматический отчёт по трудозатратам: план / факт / корректировки. Раньше – 10 часов ручной работы с ошибками. Теперь – **5 минут** без ошибок.

- **[Compliance Monitor](./compliance-monitor)** – контроль обязательств перед заказчиком. Автоматические уведомления об отклонениях. Закрыта «серая зона» требований.

- **[Release Committee Redesign](./process-redesign)** – редизайн процесса согласования инициатив. Время прохождения сокращено с 10 дней до **2 дней**, количество срывов – на **60% меньше**.
